## 📜 Description
It's a microservice project developed with [Nest](https://github.com/nestjs/nest).

## ⚙️ Installation

```bash
$ npm install
```

## 🪄 Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## 📈 Test

```bash
# unit tests
$ npm run test
```
